Section 5: Absolute Value Equations

Recall: $|x| = a \implies x = a$ or $x = -a$ (where $a \ge 0$).

10 Solved Examples

1. Solve: $|x - 5| = 3$
$x-5=3 \text{ or } -3 \implies S = \{2, 8\}$
2. Solve: $|2x + 4| = 10$
$2x+4=10 \text{ or } -10 \implies S = \{-7, 3\}$
3. Solve: $|3x - 1| = 0$
$3x-1=0 \implies S = \{\frac{1}{3}\}$
4. Solve: $|x + 7| = -2$
No solution. $S = \emptyset$
5. Solve: $|4x| = 12$
$4x=\pm 12 \implies S = \{-3, 3\}$
6. Solve: $|x - 2| + 5 = 9$
$|x-2|=4 \implies S = \{-2, 6\}$
7. Solve: $2|x| = 14$
$|x|=7 \implies S = \{-7, 7\}$
8. Solve: $|\frac{x}{2} + 1| = 4$
$x/2+1 = \pm 4 \implies S = \{-10, 6\}$
9. Solve: $|3 - x| = 5$
$3-x=\pm 5 \implies S = \{-2, 8\}$
10. Solve: $|2x - 8| = |x + 1|$
$2x-8 = \pm(x+1) \implies S = \{\frac{7}{3}, 9\}$

10 Practice Items

$S = \{-3, 5\}$

$S = \{-12, 12\}$
14. $|1 - 2x| = 7$

20. $\frac{|x|}{3} = 5$
$S = \{-15, 15\}$

Section 6: Absolute Value Inequalities

Recall: $|x| < a \implies -a < x < a$ AND $|x| > a \implies x > a \text{ or } x < -a$.

10 Solved Examples

1. Solve: $|x| < 4$
$S = (-4, 4)$
2. Solve: $|x| \ge 5$
$S = (-\infty, -5] \cup [5, +\infty)$
3. Solve: $|x - 2| \le 3$
$-3 \le x-2 \le 3 \implies S = [-1, 5]$
4. Solve: $|2x + 1| > 7$
$2x+1 > 7 \text{ or } < -7 \implies S = (-\infty, -4) \cup (3, +\infty)$
5. Solve: $|x + 5| < -1$
No solution. $S = \emptyset$
6. Solve: $|x - 1| \ge 0$
Always true. $S = \mathbb{R}$
7. Solve: $|3x - 6| < 9$
$-9 < 3x-6 < 9 \implies S = (-1, 5)$
8. Solve: $|4 - x| \le 2$
$-2 \le 4-x \le 2 \implies S = [2, 6]$
9. Solve: $2|x + 3| \le 10$
$|x+3| \le 5 \implies S = [-8, 2]$
10. Solve: $|\frac{x}{3}| > 2$
$S = (-\infty, -6) \cup (6, +\infty)$

10 Practice Items

11. $|x + 2| < 8$
$S = (-10, 6)$
12. $|x - 6| \ge 10$
$S = (-\infty, -4] \cup [16, +\infty)$
13. $|2x - 4| \le 6$
$S = [-1, 5]$
14. $|5x| > 15$
$S = (-\infty, -3) \cup (3, +\infty)$
15. $|1 - 3x| < 4$
$S = (-1, \frac{5}{3})$
16. $|x + 10| \ge 0$
$S = \mathbb{R}$
17. $|x - 5| < -2$
$S = \emptyset$
18. $|\frac{x}{2}| \le 1$
$S = [-8, -4]$
19. $3|x| + 2 < 11$
$S = (-3, 3)$
20. $|7 - x| \ge 3$
$S = (-\infty, 4] \cup [10, +\infty)$
