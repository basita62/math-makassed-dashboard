Section 1: Common Factors & Zero Product Property

10 Solved Examples (With Steps)

1. $x^2 + 4x = 0$
Factor $x$: $x(x + 4) = 0$ → $x = 0$ or $x = -4$
2. $3x^2 - 12x = 0$
Factor $3x$: $3x(x - 4) = 0$ → $x = 0$ or $x = 4$
3. $(x+1)(x+3) + (x+1)(2x+5) = 0$
Factor $(x+1)$: $(x+1)[(x+3) + (2x+5)] = 0$ → $(x+1)(3x+8) = 0$ → $x = -1$ or $x = -8/3$
4. $5x^2 = 10x$
$5x^2 - 10x = 0 \rightarrow 5x(x - 2) = 0 \rightarrow x = 0, 2$
5. $(x-2)(2x+1) - (x-2)(x+5) = 0$
$(x-2)(2x+1-x-5) = 0 \rightarrow (x-2)(x-4) = 0 \rightarrow x = 2, 4$
6. $x(x-5) + 3(x-5) = 0 \rightarrow (x-5)(x+3) = 0 \rightarrow x = 5, -3$
7. $7x^2 + 21x = 0 \rightarrow 7x(x+3) = 0 \rightarrow x = 0, -3$
8. $(2x-1)(x+4) = 0 \rightarrow x = 1/2, -4$
9. $x^2 - x = 0 \rightarrow x(x-1) = 0 \rightarrow x = 0, 1$
10. $4x^2 + 8x = 0 \rightarrow 4x(x+2) = 0 \rightarrow x = 0, -2$

Section 2: Solving Using Identities

10 Solved Examples

1. $x^2 - 16 = 0 \rightarrow (x-4)(x+4) = 0 \rightarrow x = 4, -4$
2. $x^2 - 6x + 9 = 0 \rightarrow (x-3)^2 = 0 \rightarrow x = 3$
3. $4x^2 - 25 = 0 \rightarrow (2x-5)(2x+5) = 0 \rightarrow x = \pm 2.5$
4. $x^2 + 10x + 25 = 0 \rightarrow (x+5)^2 = 0 \rightarrow x = -5$
5. $9x^2 - 1 = 0 \rightarrow (3x-1)(3x+1) = 0 \rightarrow x = \pm 1/3$
6. $x^2 - 12x + 36 = 0 \rightarrow (x-6)^2 = 0 \rightarrow x = 6$
7. $49 - x^2 = 0 \rightarrow (7-x)(7+x) = 0 \rightarrow x = \pm 7$
8. $4x^2 + 4x + 1 = 0 \rightarrow (2x+1)^2 = 0 \rightarrow x = -0.5$
9. $x^2 - 100 = 0 \rightarrow x = \pm 10$
10. $25x^2 - 20x + 4 = 0 \rightarrow (5x-2)^2 = 0 \rightarrow x = 0.4$

Section 3: Mixed Zero Product Challenges

10 Solved Examples

1. $(x-5)^2 - 9 = 0 \rightarrow (x-5-3)(x-5+3) = 0 \rightarrow (x-8)(x-2)=0 \rightarrow x=8, 2$
2. $x^3 - x = 0 \rightarrow x(x^2-1) = 0 \rightarrow x(x-1)(x+1) = 0 \rightarrow x = 0, 1, -1$
3. $(2x+3)^2 = (x-1)^2 \rightarrow (2x+3)^2 - (x-1)^2 = 0 \rightarrow x = -4, -2/3$
4. $x^2 - 4 + (x-2)(x+5) = 0 \rightarrow (x-2)(x+2) + (x-2)(x+5) = 0 \rightarrow x=2, -3.5$
5. $3x^2 - 75 = 0 \rightarrow 3(x^2-25) = 0 \rightarrow x = \pm 5$
6. $x^2(x+2) - 4(x+2) = 0 \rightarrow (x+2)(x-2)(x+2) \rightarrow x = \pm 2$
7. $x^2 - 2x + 1 = 4 \rightarrow (x-1)^2 - 2^2 = 0 \rightarrow x = 3, -1$
8. $x^2 = 3x \rightarrow x(x-3)=0 \rightarrow x=0, 3$
9. $(x+1)^2 - 25 = 0 \rightarrow x=4, -6$
10. $x^2 - 6x = -9 \rightarrow (x-3)^2=0 \rightarrow x=3$

Section 4: Harder Equations (Advanced Factoring)

10 Solved Examples

1. $x^4 - 81 = 0 \rightarrow (x^2-9)(x^2+9)=0 \rightarrow x = \pm 3$
2. $x^3 - 4x^2 - 9x + 36 = 0 \rightarrow (x-4)(x^2-9)=0 \rightarrow x = 4, \pm 3$
3. $2x^3 = 50x \rightarrow 2x(x^2-25)=0 \rightarrow x = 0, \pm 5$
4. $(x^2-1)^2 - 9 = 0 \rightarrow (x^2-4)(x^2+2)=0 \rightarrow x = \pm 2$
5. $x^2(x-1) - 4(x-1) = 0 \rightarrow (x-1)(x^2-4)=0 \rightarrow x = 1, \pm 2$
6. $x^4 - 5x^2 + 4 = 0 \rightarrow (x^2-4)(x^2-1)=0 \rightarrow x = \pm 2, \pm 1$
7. $3x^3 - 12x^2 + 12x = 0 \rightarrow 3x(x-2)^2=0 \rightarrow x = 0, 2$
8. $(x+1)^2 - (2x-3)^2 = 0 \rightarrow x = 4, 2/3$
9. $x^3 + x^2 - x - 1 = 0 \rightarrow (x+1)^2(x-1)=0 \rightarrow x = \pm 1$
10. $16x^4 - 1 = 0 \rightarrow (4x^2-1)(4x^2+1)=0 \rightarrow x = \pm 0.5$
