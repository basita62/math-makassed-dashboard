Section 1: Angles: Finding the unknown

Right, Straight and full angles - Perpendicular lines

Solved Examples

EX. 1

$\text{Calculate the measure of the angle marked } x.$

Solution:

$x + 90^\circ + 20^\circ = 180^\circ \text{ (sum of angles of a triangle)} \\ x= 180^\circ - 90^\circ - 20^\circ \\ x = 70^\circ$

EX. 2

$\text{Calculate the measure of the angle marked } x.$

Solution:

$x + 50^\circ + 90^\circ + 10^\circ = 180^\circ \text{ (straight angle)} \\ x= 180^\circ - 50^\circ - 90^\circ - 10^\circ \\ x = 30^\circ$

EX. 3

$\text{Given: (DA) is perpendicular to (DC), } \\ \widehat{BDE} \text{ is straight, } \widehat{ADE} = 140^\circ.$

$\text{Calculate the measure of the angle } \widehat{CDE}.$

Solution:

$\widehat{CDE} + 90^\circ + 140^\circ = 360^\circ \text{ (full angle)} \\ \widehat{CDE} = 360^\circ - 90^\circ - 140^\circ \\ \widehat{CDE} = 130^\circ$

EX. 4

$\text{Given: (PG) is perpendicular to (PK), } \\ \widehat{GPJ} = 65^\circ \text{ ; } \widehat{KPH} = 40^\circ.$

$\text{Calculate the measure of the angle marked } x.$

Solution:

$\widehat{HPG} = 90^\circ - 40^\circ = 50^\circ \\ x = 65^\circ - 50^\circ \\ x = 15^\circ$

EX. 5

$\text{Given: } \widehat{XOY} = 80^\circ \text{ ; } [OT) \text{ bisects } \widehat{XOY}.$

$\text{Calculate the measure of the angle marked } x.$

Solution:

$\widehat{XOT} = \frac{80^\circ}{2} = 40^\circ \\ x = 40^\circ - 18^\circ = 22^\circ$

Section 2: Triangles: Finding the unknown

Angle Sum Property and Exterior Angle Theorem - Special triangles

Solved Examples

EX. 1

$\text{Calculate the measure of the angle marked } x.$

Solution:

$\widehat{ADB} = \widehat{DAB} = 25^\circ \text{ (base angles of an isosceles triangle)} \\ \widehat{DBC} = 25^\circ + 25^\circ = 50^\circ \text{ (exterior angle of a triangle)} \\ x + 50^\circ + 70^\circ = 180^\circ \text{ (sum of angles of a triangle)} \\ x= 180^\circ - 50^\circ - 70^\circ \\ x = 60^\circ$

EX. 2

$\text{Calculate the measure of the angle marked } x.$

Solution:

$\widehat{ACB} = 180^\circ - 135^\circ = 45^\circ \\ x = 70^\circ + 45^\circ \text{ (exterior angle of a triangle)} \\ x = 115^\circ$

EX. 3

$\text{Calculate the measure of the angle marked } x.$

Solution:

$\widehat{EBD} = \frac{180^\circ - 116^\circ}{2} = 32^\circ \text{ (base angle in an isosceles triangle)} \\ \widehat{BDC} = 180^\circ - 116^\circ = 64^\circ \\ \widehat{CBD} = \widehat{BDC} = 64^\circ \text{ (base angles of an isosceles triangle)} \\ x = 180^\circ - 64^\circ - 32^\circ = 84^\circ$

EX. 4

$\text{Calculate the measure of the angle marked } y.$

Solution:

$\widehat{PSQ} = \widehat{SPQ} = 25^\circ \text{ (base angles in an isosceles triangle)} \\ \widehat{SQR} = 25^\circ + 25^\circ = 50^\circ \\ \widehat{SRQ} = \widehat{RSQ} = y \text{ (base angles in an isosceles triangle)} \\y + y + 50^\circ = 180^\circ \\ 2y = 180^\circ - 50^\circ \\ 2y = 130^\circ \\ y = 65^\circ$

EX. 5

$\text{Calculate the measure of the angle marked } x.$

Solution:

$\widehat{DBC} = \widehat{DCB} = \frac{180^\circ - 120^\circ}{2} = 30^\circ \text{ (base angle in an isosceles triangle)} \\ \widehat{DBA} = 180^\circ - 30^\circ = 150^\circ \\ y + 150^\circ + 108^\circ + 54^\circ = 360^\circ \text{ (angles in a quadrilateral: 2 traingles)} \\y = 360^\circ - 150^\circ - 108^\circ - 54^\circ \\ y = 48^\circ$
