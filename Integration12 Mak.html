2. Substitution Method

4. Integration by Partial Fractions

5 Solved Examples

5. $\int \frac{5x-3}{x^2-2x-3} dx = 3\ln|x-3| + 2\ln|x+1| + C$

Definite Integrals

20 Diverse Solved Examples

1. $\int_1^3 x^2 dx = [\frac{x^3}{3}]_1^3 = \frac{26}{3}$
2. $\int_0^1 e^{2x} dx = [\frac{1}{2}e^{2x}]_0^1 = \frac{e^2-1}{2}$
3. $\int_1^4 \sqrt{x} dx = [\frac{2}{3}x^{3/2}]_1^4 = \frac{14}{3}$
4. $\int_2^5 \frac{1}{x-1} dx = [\ln|x-1|]_2^5 = \ln 4$
5. $\int_0^2 (3x^2-2x) dx = [x^3-x^2]_0^2 = 4$
6. $\int_1^e \frac{\ln x}{x} dx = [\frac{(\ln x)^2}{2}]_1^e = 0.5$
7. $\int_0^1 \frac{x}{x^2+1} dx = [\frac{1}{2}\ln(x^2+1)]_0^1 = \frac{\ln 2}{2}$
8. $\int_1^2 \frac{1}{x^3} dx = [-\frac{1}{2x^2}]_1^2 = \frac{3}{8}$
9. $\int_0^{\ln 2} e^x dx = [e^x]_0^{\ln 2} = 2-1 = 1$
10. $\int_0^1 (x+1)^3 dx = [\frac{(x+1)^4}{4}]_0^1 = \frac{15}{4}$
11. $\int_1^2 \frac{e^{1/x}}{x^2} dx = [-e^{1/x}]_1^2 = e - \sqrt{e}$
12. $\int_0^1 xe^x dx = [xe^x - e^x]_0^1 = 1$
13. $\int_1^4 \frac{1}{\sqrt{x}} dx = [2\sqrt{x}]_1^4 = 2$
14. $\int_0^2 \sqrt{4x+1} dx = [\frac{1}{6}(4x+1)^{3/2}]_0^2 = \frac{13}{3}$
15. $\int_2^3 \frac{1}{x^2-1} dx = \frac{1}{2}\ln(3/2)$
16. $\int_1^e \ln x dx = [x\ln x - x]_1^e = 1$
17. $\int_0^1 \frac{x^2}{x^3+1} dx = \frac{\ln 2}{3}$
18. $\int_2^4 \frac{1}{x} dx = \ln 2$
19. $\int_0^1 e^{-x} dx = 1 - e^{-1}$
20. $\int_1^2 (x^2 + \frac{1}{x^2}) dx = \frac{17}{6}$

20 Solved Area Variations

Ex 6.2: Between Two Curves ($y=x$ and $y=x^2$).
