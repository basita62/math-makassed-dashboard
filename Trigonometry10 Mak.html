3. Trigonometric Calculations

Solved Exercises

1. Evaluate: \(A = 2\sqrt{3} \sin \frac{2\pi}{3} + 2 \sin \frac{5\pi}{6} - 4\sqrt{2} \sin \frac{5\pi}{4} + \sin \frac{3\pi}{2}\)

▶ Click to reveal solution

\[ 2\sqrt{3} \sin \frac{2\pi}{3} = 2\sqrt{3} \left(\frac{\sqrt{3}}{2}\right) = 3 \]

\[ 2\sin \frac{5\pi}{6} = 2 \left(\frac{1}{2}\right) = 1 \]

\[ -4\sqrt{2} \sin \frac{5\pi}{4} = -4\sqrt{2} \left(-\frac{\sqrt{2}}{2}\right) = 4 \]

\[ \sin \frac{3\pi}{2} = -1 \]

Result: \(A = 3 + 1 + 4 - 1 = 7\)

2. Evaluate: \(B = \sin \frac{3\pi}{11} - \cos \frac{5\pi}{22} - \sin \frac{\pi}{11} + \sin \frac{10\pi}{11}\)

▶ Click to reveal solution

Note that \(\sin \frac{10\pi}{11} = \sin\left(\pi - \frac{\pi}{11}\right) = \sin \frac{\pi}{11}\)

Note that \(\cos \frac{5\pi}{22} = \sin\left(\frac{\pi}{2} - \frac{5\pi}{22}\right) = \sin \frac{6\pi}{22} = \sin \frac{3\pi}{11}\)

Substituting back: \[ B = \sin \frac{3\pi}{11} - \sin \frac{3\pi}{11} - \sin \frac{\pi}{11} + \sin \frac{\pi}{11} \]

Result: \(B = 0\)

3. Evaluate: \(\sin\left(\frac{2\pi}{3}\right) + \cos\left(\frac{19\pi}{6}\right) + \tan\left(-\frac{15\pi}{4}\right)\)

▶ Click to reveal solution

\[ \sin\left(\frac{2\pi}{3}\right) = \sin\left(\pi - \frac{\pi}{3}\right) = \frac{\sqrt{3}}{2} \]

\[ \cos\left(\frac{19\pi}{6}\right) = \cos\left(3\pi + \frac{\pi}{6}\right) = -\cos\left(\frac{\pi}{6}\right) = -\frac{\sqrt{3}}{2} \]

\[ \tan\left(-\frac{15\pi}{4}\right) = -\tan\left(4\pi - \frac{\pi}{4}\right) = -(-\tan\frac{\pi}{4}) = 1 \]

Result: \(\frac{\sqrt{3}}{2} - \frac{\sqrt{3}}{2} + 1 = 1\)

4. Evaluate: \(\sin 150^\circ + \cos 240^\circ + \tan(-315^\circ)\)

▶ Click to reveal solution

\[ \sin 150^\circ = \sin(180^\circ - 30^\circ) = \frac{1}{2} \]

\[ \cos 240^\circ = \cos(180^\circ + 60^\circ) = -\frac{1}{2} \]

\[ \tan(-315^\circ) = -\tan(315^\circ) = -\tan(360^\circ - 45^\circ) = 1 \]

Result: \(\frac{1}{2} - \frac{1}{2} + 1 = 1\)

5. Evaluate: \(\sin^2 5^\circ + \sin^2 85^\circ + \tan 10^\circ \cdot \cot 190^\circ\)

▶ Click to reveal solution

Using \(\sin 85^\circ = \cos(90^\circ - 85^\circ) = \cos 5^\circ\):

\[ \sin^2 5^\circ + \cos^2 5^\circ = 1 \]

Using \(\cot 190^\circ = \cot(180^\circ + 10^\circ) = \cot 10^\circ\):

\[ \tan 10^\circ \cdot \cot 10^\circ = 1 \]

Result: \(1 + 1 = 2\)

6. Simplify: \(\sin^2\left(\frac{\pi}{3} - x\right) + \sin^2\left(\frac{\pi}{6} + x\right)\)

▶ Click to reveal solution

Notice that the angles are complementary:

\[ \left(\frac{\pi}{3} - x\right) + \left(\frac{\pi}{6} + x\right) = \frac{2\pi + \pi}{6} = \frac{\pi}{2} \]

Therefore, \(\sin\left(\frac{\pi}{6} + x\right) = \cos\left[\frac{\pi}{2} - \left(\frac{\pi}{6} + x\right)\right] = \cos\left(\frac{\pi}{3} - x\right)\)

Substituting back:

\[ \sin^2\left(\frac{\pi}{3} - x\right) + \cos^2\left(\frac{\pi}{3} - x\right) = 1 \]

Result: 1

7. Evaluate: \( \cos^2 25^\circ - \cos^2 45^\circ + \cos^2 65^\circ - \frac{1}{2} \tan^2 10^\circ \cdot \tan^2 80^\circ \)

▶ Click to reveal solution

Using complementary angles: \( \cos 65^\circ = \sin(90^\circ - 65^\circ) = \sin 25^\circ \)

\[ \cos^2 25^\circ + \cos^2 65^\circ = \cos^2 25^\circ + \sin^2 25^\circ = 1 \]

Using \( \tan 80^\circ = \cot 10^\circ \):

\[ \tan^2 10^\circ \cdot \tan^2 80^\circ = (\tan 10^\circ \cdot \cot 10^\circ)^2 = 1^2 = 1 \]

Known value: \( \cos^2 45^\circ = \left(\frac{\sqrt{2}}{2}\right)^2 = \frac{1}{2} \)

Substituting all values into the expression:

\[ 1 - \frac{1}{2} - \frac{1}{2}(1) = 1 - \frac{1}{2} - \frac{1}{2} \]

Result: 0

8. Evaluate: \( -2 \sin\frac{23\pi}{6} + 2\sqrt{2} \cos\frac{23\pi}{4} + \sqrt{3} \tan\frac{23\pi}{3} \)

▶ Click to reveal solution

Reducing angles by their periods (\( 2\pi \)):

\[ \sin\frac{23\pi}{6} = \sin\left(4\pi - \frac{\pi}{6}\right) = -\sin\frac{\pi}{6} = -\frac{1}{2} \]

\[ \cos\frac{23\pi}{4} = \cos\left(6\pi - \frac{\pi}{4}\right) = \cos\frac{\pi}{4} = \frac{\sqrt{2}}{2} \]

\[ \tan\frac{23\pi}{3} = \tan\left(8\pi - \frac{\pi}{3}\right) = -\tan\frac{\pi}{3} = -\sqrt{3} \]

Substituting back:

\[ -2\left(-\frac{1}{2}\right) + 2\sqrt{2}\left(\frac{\sqrt{2}}{2}\right) + \sqrt{3}(-\sqrt{3}) \]

\[ 1 + 2 - 3 = 0 \]

Result: 0

9. Evaluate: \( \tan \frac{\pi}{4} + \tan \frac{3\pi}{4} + \tan \frac{5\pi}{4} \)

▶ Click to reveal solution

Evaluate each term using the unit circle or periodicity:

\[ \tan \frac{\pi}{4} = 1 \]

\[ \tan \frac{3\pi}{4} = \tan\left(\pi - \frac{\pi}{4}\right) = -\tan \frac{\pi}{4} = -1 \]

\[ \tan \frac{5\pi}{4} = \tan\left(\pi + \frac{\pi}{4}\right) = \tan \frac{\pi}{4} = 1 \]

Summing the values:

\[ 1 + (-1) + 1 = 1 \]

Result: 1

10. Evaluate: \( 2\cos\left(\frac{2\pi}{3}\right) + \sqrt{3}\tan\left(\frac{4\pi}{3}\right) - \sin\left(\frac{7\pi}{6}\right) \)

▶ Click to reveal solution

Evaluate each term individually:

\[ 2\cos\left(\frac{2\pi}{3}\right) = 2\left(-\frac{1}{2}\right) = -1 \]

\[ \sqrt{3}\tan\left(\frac{4\pi}{3}\right) = \sqrt{3}(\sqrt{3}) = 3 \]

\[ -\sin\left(\frac{7\pi}{6}\right) = -\left(-\frac{1}{2}\right) = 0.5 \]

Summing the values:

\[ -1 + 3 + 0.5 = 2.5 \]

Result: 2.5

4. Simplifying Trigonometric Expressions

1. Simplify: \( A = \frac{-\cos(7\pi - x) - \sin\left(\frac{3\pi}{2} - x\right)}{2\cos\left(\frac{5\pi}{2} - x\right)} \)

▶ Click to reveal solution

Step 1: Simplify Numerator

\[ -\cos(7\pi - x) = -\cos(\pi - x) = -(-\cos x) = \cos x \]

\[ \sin\left(\frac{3\pi}{2} - x\right) = -\cos x \]

Numerator total: \( \cos x - (-\cos x) = 2\cos x \)

Step 2: Simplify Denominator

\[ 2\cos\left(\frac{5\pi}{2} - x\right) = 2\cos\left(\frac{\pi}{2} - x\right) = 2\sin x \]

Result: \( A = \frac{2\cos x}{2\sin x} = \cot x \)

2. Simplify: \( B = \sin(-21\pi + x) + \sin\left(\frac{5\pi}{2} - x\right) + \cos(-3\pi - x) - \cos\left(-\frac{3\pi}{2} - x\right) \)

▶ Click to reveal solution

Step 1: Reduce each term using periodicity

  • \( \sin(-21\pi + x) = \sin(-\pi + x) = -\sin x \)
  • \( \sin\left(\frac{5\pi}{2} - x\right) = \cos x \)
  • \( \cos(-3\pi - x) = -\cos x \)
  • \( \cos\left(-\frac{3\pi}{2} - x\right) = \sin x \)

Step 2: Combine

\[ B = (-\sin x) + (\cos x) + (-\cos x) - (\sin x) \]

Result: \( B = -2\sin x \)

3. Simplify: \( E = \frac{\sin\left(\frac{3\pi}{2} + x\right) \cdot \cot\left(\frac{5\pi}{2} + x\right)}{\cot\left(x + \frac{3\pi}{2}\right)} \)

▶ Click to reveal solution

Step 1: Simplify each component

  • \( \sin\left(\frac{3\pi}{2} + x\right) = -\cos x \)
  • \( \cot\left(\frac{5\pi}{2} + x\right) = \cot\left(\frac{\pi}{2} + x\right) = -\tan x \)
  • \( \cot\left(x + \frac{3\pi}{2}\right) = -\tan x \)

Step 2: Substitute and simplify

\[ E = \frac{(-\cos x) \cdot (-\tan x)}{-\tan x} \]

Final Answer: \( E = -\cos x \)

4. Simplify: \( F = \cos(x - 7\pi) - \sin\left(x - \frac{27\pi}{2}\right) + \cos(x + 3\pi) + \sin\left(x + \frac{3\pi}{2}\right) \)

▶ Click to reveal solution

Step 1: Reduce each term using periodicity (\(2\pi\))

  • \( \cos(x - 7\pi) = \cos(x - \pi) = \mathbf{-\cos x} \)
  • \( \sin\left(x - \frac{27\pi}{2}\right) = \sin\left(x - \frac{3\pi}{2}\right) = \mathbf{\cos x} \)
  • \( \cos(x + 3\pi) = \cos(x + \pi) = \mathbf{-\cos x} \)
  • \( \sin\left(x + \frac{3\pi}{2}\right) = \mathbf{-\cos x} \)

Step 2: Combine terms

\( F = (-\cos x) - (\cos x) + (-\cos x) + (-\cos x) \)

Final Answer: \( F = -4\cos x \)

5. Simplify: \( C = \frac{\tan\left(\frac{13\pi}{2} - x\right) \cdot \cot(44\pi - x) \cdot \sin(17\pi - x)}{\sin\left(\frac{3\pi}{2} - x\right) \cdot \cos(9\pi - x)} \)

▶ Click to reveal solution

Step 1: Reduce each term using periodicity

  • \( \tan\left(\frac{13\pi}{2} - x\right) = \tan\left(\frac{\pi}{2} - x\right) = \cot x \)
  • \( \cot(44\pi - x) = \cot(-x) = -\cot x \)
  • \( \sin(17\pi - x) = \sin(\pi - x) = \sin x \)
  • \( \sin\left(\frac{3\pi}{2} - x\right) = -\cos x \)
  • \( \cos(9\pi - x) = \cos(\pi - x) = -\cos x \)

Step 2: Substitute and simplify

\[ C = \frac{\cot x \cdot (-\cot x) \cdot \sin x}{(-\cos x) \cdot (-\cos x)} = \frac{-\cot^2 x \cdot \sin x}{\cos^2 x} \]

Converting to sine and cosine: \[ C = \frac{-\frac{\cos^2 x}{\sin^2 x} \cdot \sin x}{\cos^2 x} = -\frac{1}{\sin x} \]

Final Answer: \( C = -\frac{1}{\sin x} \)

6. Simplify: \( D = \cos^2 a + \cos^2\left(\frac{\pi}{2} - a\right) + \cos^2\left(\frac{\pi}{2} + a\right) + \cos^2(\pi - a) \)

▶ Click to reveal solution

Step 1: Apply reduction identities

  • \( \cos^2\left(\frac{\pi}{2} - a\right) = (\sin a)^2 = \sin^2 a \)
  • \( \cos^2\left(\frac{\pi}{2} + a\right) = (-\sin a)^2 = \sin^2 a \)
  • \( \cos^2(\pi - a) = (-\cos a)^2 = \cos^2 a \)

Step 2: Combine using Pythagorean Identity \( \sin^2 a + \cos^2 a = 1 \)

\[ D = \cos^2 a + \sin^2 a + \sin^2 a + \cos^2 a \]

\[ D = (\cos^2 a + \sin^2 a) + (\sin^2 a + \cos^2 a) = 1 + 1 \]

Final Answer: D = 2

5. Trigonometric Reductions and Quadrant Analysis

In this section, we determine the exact values of trigonometric functions by identifying the specific quadrant based on given intervals and signs.

1. Given \(\sin \alpha = -\frac{2}{3}\) and \(\alpha \in [\frac{\pi}{2}; \frac{3\pi}{2}]\), calculate \(\cos \alpha\) and \(\tan \alpha\).

▶ Click to reveal solution

Step 1: Quadrant Check. Since \(\sin \alpha\) is negative in \([\frac{\pi}{2}, \frac{3\pi}{2}]\), \(\alpha\) is in Quadrant III. Thus, \(\cos \alpha < 0\) and \(\tan \alpha > 0\).

Step 2: Find \(\cos \alpha\). Using \(\cos^2 \alpha = 1 - \sin^2 \alpha\):
\[ \cos^2 \alpha = 1 - \left(-\frac{2}{3}\right)^2 = 1 - \frac{4}{9} = \frac{5}{9} \]

Step 3: Final Values.
\[ \cos \alpha = \mathbf{-\frac{\sqrt{5}}{3}}, \quad \tan \alpha = \frac{-2/3}{-\sqrt{5}/3} = \mathbf{\frac{2\sqrt{5}}{5}} \]

2. Given \(\cos \alpha = \frac{\sqrt{2}}{4}\) and \(\alpha \in [-\pi; 0]\), calculate \(\sin \alpha\) and \(\tan \alpha\).

▶ Click to reveal solution

Step 1: Quadrant Check. \(\alpha \in [-\pi, 0]\) and \(\cos \alpha > 0\) places \(\alpha\) in Quadrant IV. Thus, \(\sin \alpha < 0\) and \(\tan \alpha < 0\).

Step 2: Find \(\sin \alpha\).
\[ \sin^2 \alpha = 1 - \left(\frac{\sqrt{2}}{4}\right)^2 = 1 - \frac{2}{16} = \frac{7}{8} \]

Step 3: Final Values.
\[ \sin \alpha = -\sqrt{\frac{7}{8}} = \mathbf{-\frac{\sqrt{14}}{4}}, \quad \tan \alpha = \cfrac{-\frac{\sqrt{14}}{4}}{\frac{\sqrt{2}}{4}} = \mathbf{-\sqrt{7}} \]

3. Given \(\tan \alpha = 2\) and \(\alpha \in [\frac{\pi}{2}; \frac{3\pi}{2}]\), calculate \(\cos \alpha\) and \(\sin \alpha\).

▶ Click to reveal solution

Step 1: Quadrant Check. \(\tan \alpha > 0\) in \([\frac{\pi}{2}, \frac{3\pi}{2}]\) means \(\alpha\) is in Quadrant III. Thus, \(\cos \alpha < 0\) and \(\sin \alpha < 0\).

Step 2: Find \(\cos \alpha\). Using \(1 + \tan^2 \alpha = \frac{1}{\cos^2 \alpha}\):
\[ 1 + (2)^2 = \frac{1}{\cos^2 \alpha} \implies \cos^2 \alpha = \frac{1}{5} \]

Step 3: Final Values.
\[ \cos \alpha = \mathbf{-\frac{\sqrt{5}}{5}}, \quad \sin \alpha = \tan \alpha \cdot \cos \alpha = \mathbf{-\frac{2\sqrt{5}}{5}} \]

4. Given \( 9 \cos^2 \alpha \tan^2 \alpha = 5 \) for \( \alpha \in [-\pi; -\frac{\pi}{2}] \), calculate \( \sin \alpha \) and \( \cos \alpha \).

▶ Click to reveal solution

Step 1: Simplify the given equation

Using the identity \( \tan \alpha = \frac{\sin \alpha}{\cos \alpha} \):

\[ 9 \cos^2 \alpha \left( \frac{\sin^2 \alpha}{\cos^2 \alpha} \right) = 5 \implies 9 \sin^2 \alpha = 5 \]

Step 2: Solve for Sine

For \( \alpha \in [-\pi; -\frac{\pi}{2}] \) (Quadrant III), \( \sin \alpha \) must be negative:

\[ \sin^2 \alpha = \frac{5}{9} \implies \mathbf{\sin \alpha = -\frac{\sqrt{5}}{3}} \]

Step 3: Solve for Cosine

In Quadrant III, \( \cos \alpha \) is also negative:

\[ \cos^2 \alpha = 1 - \frac{5}{9} = \frac{4}{9} \implies \mathbf{\cos \alpha = -\frac{2}{3}} \]

Final Results:

\(\sin \alpha = -\frac{\sqrt{5}}{3}\), \(\cos \alpha = -\frac{2}{3}\)

5. Given \( \sin \alpha = -\frac{\sqrt{2}}{6} \) for \( -\frac{\pi}{2} \leq \alpha \leq 0 \), calculate the exact value of \( \cos \alpha \) and \( \tan \alpha \).

▶ Click to reveal solution

Step 1: Quadrant Analysis

The interval \( [-\frac{\pi}{2}, 0] \) places \(\alpha\) in Quadrant IV. In this quadrant, \( \cos \alpha \) is positive (+) and \( \tan \alpha \) is negative (-).

Step 2: Calculate \(\cos \alpha\)

Using the identity \( \cos^2 \alpha = 1 - \sin^2 \alpha \):

\[ \cos^2 \alpha = 1 - \left(-\frac{\sqrt{2}}{6}\right)^2 = 1 - \frac{2}{36} = \frac{17}{18} \]

Since \(\cos \alpha\) is positive: \[ \cos \alpha = \sqrt{\frac{17}{18}} = \frac{\sqrt{17}}{3\sqrt{2}} = \mathbf{\frac{\sqrt{34}}{6}} \]

Step 3: Calculate \(\tan \alpha\)

\[ \tan \alpha = \frac{\sin \alpha}{\cos \alpha} = \frac{-\sqrt{2}/6}{\sqrt{34}/6} = -\frac{\sqrt{2}}{\sqrt{34}} = -\frac{1}{\sqrt{17}} \]

\[ \tan \alpha = \mathbf{-\frac{\sqrt{17}}{17}} \]

Final Exact Values:

\(\cos \alpha = \frac{\sqrt{34}}{6}\), \(\tan \alpha = -\frac{\sqrt{17}}{17}\)
