Section 3: Characteristics of direct similitude

10 Solved Examples (Find $k$, $\theta$, $\Omega$)

1. $z' = (2 + 2i)z + 4$
Solution:
$a = 2 + 2i$, $b = 4$
$k = |a| = 2\sqrt{2}$, $\theta = \arg(a) = \cfrac{\pi}{4}$
$\omega = \cfrac{b}{1-a} = \cfrac{4}{-1-2i} = -\cfrac{4}{5} + \cfrac{8}{5}i$
$\mathbf{k=2\sqrt{2}}$, $\mathbf{\theta=\cfrac{\pi}{4}}$, $\mathbf{\Omega\left(-\cfrac{4}{5}, \cfrac{8}{5}\right)}$
2. $z' = (1 - i)z + 3i$
Solution:
$k = |1 - i| = \sqrt{2}$, $\theta = -\cfrac{\pi}{4}$
$\omega = \cfrac{3i}{i} = 3$
$\mathbf{k=\sqrt{2}}$, $\mathbf{\theta=-\cfrac{\pi}{4}}$, $\mathbf{\Omega\left(3, 0\right)}$
3. $z' = (\sqrt{3} - i)z - \sqrt{3}$
Solution:
$k = |\sqrt{3} - i| = 2$, $\theta = -\cfrac{\pi}{6}$
$\omega = \cfrac{-\sqrt{3}}{1-\sqrt{3}+i} = \cfrac{3-\sqrt{3}+\sqrt{3}i}{5-2\sqrt{3}}$
$\mathbf{k=2}$, $\mathbf{\theta=-\cfrac{\pi}{6}}$, $\mathbf{\Omega\left(\cfrac{3-\sqrt{3}}{5-2\sqrt{3}}, \cfrac{\sqrt{3}}{5-2\sqrt{3}}\right)}$
4. $z' = (3i)z + 1 + i$
Solution:
$k = |3i| = 3$, $\theta = \cfrac{\pi}{2}$
$\omega = \cfrac{1+i}{1-3i} = -\cfrac{1}{5} + \cfrac{2}{5}i$
$\mathbf{k=3}$, $\mathbf{\theta=\cfrac{\pi}{2}}$, $\mathbf{\Omega\left(-\cfrac{1}{5}, \cfrac{2}{5}\right)}$
5. $z' = (-4)z + 8 - 4i$
Solution:
$k = |-4| = 4$, $\theta = \pi$
$\omega = \cfrac{8-4i}{5} = \cfrac{8}{5} - \cfrac{4}{5}i$
$\mathbf{k=4}$, $\mathbf{\theta=\pi}$, $\mathbf{\Omega\left(\cfrac{8}{5}, -\cfrac{4}{5}\right)}$
6. $z' = \left(\cfrac{1}{2} - \cfrac{\sqrt{3}}{2}i\right)z + \cfrac{1}{2}$
Solution:
$k = |a| = 1$, $\theta = -\cfrac{\pi}{3}$
$\omega = \cfrac{1/2}{1-a} = \cfrac{1}{4} - \cfrac{\sqrt{3}}{4}i$
$\mathbf{k=1}$, $\mathbf{\theta=-\cfrac{\pi}{3}}$, $\mathbf{\Omega\left(\cfrac{1}{4}, -\cfrac{\sqrt{3}}{4}\right)}$
7. $z' = (-1 + i)z + 2$

Solution:
$k = |a| = 1$, $\theta = -\cfrac{2\pi}{3}$

9. $z' = \left(\cfrac{\sqrt{2}}{2} + \cfrac{\sqrt{2}}{2}i\right)z - i$
