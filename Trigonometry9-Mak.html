2. Solving Triangles

Solved Examples

Detailed solutions.

1. Find Hypotenuse (Cosine)

Given: A right triangle ABC (right angle at C) with θB = 25°, the adjacent side BC = 7 units.
Calculate: The hypotenuse x = AB.

Step 1: Identify the trigonometric ratio We are given the adjacent side and need to find the hypotenuse. The cosine function relates these values:

$cos(θ) = \cfrac{Adjacent}{Hypotenuse}$

Step 2: Substitute the known values and solve for x

$cos(25°) = \cfrac{7}{x}$

Rearranging the equation to solve for x:

$x=\cfrac{7}{cos(25°)}$

Step 3: Calculate the numerical value Using a calculator:

x ≈ 7.721

**Answer:** The value of x is approximately 7.72 units.

2. Find Opposite (Sine)

Given: A right triangle ABC (right angle at C) with θB = 40°, the hypotenuse AB = 12 units.
Calculate: The opposite side x = AC.

Step 1: Identify the trigonometric ratio We are given the hypotenuse and need to find the opposite side. The sine function relates these values:

$sin(θ) = \cfrac{Opposite}{Hypotenuse}$

Step 2: Substitute the known values and solve for x

$sin(40°) = \cfrac{x}{12}$

Rearranging the equation to solve for x:

x = 12 × sin(40°)

Step 3: Calculate the numerical value Using a calculator:

x ≈ 7.713

**Answer:** The value of x is approximately 7.71 units.

3. Find Adjacent (Tangent)

Given: A right triangle ABC (right angle at C) with θB = 60°, the opposite side AC = 10 units.
Calculate: The adjacent side x = BC.

Step 1: Identify the trigonometric ratio We are given the opposite side and need to find the adjacent side. The tangent function relates these values:

$tan(θ) = \cfrac{Opposite}{Adjacent}$

Step 2: Substitute the known values and solve for x

$tan(60°) = \cfrac{10}{x}$

Rearranging the equation to solve for x:

x = $\cfrac{10}{tan(60°)}$

Step 3: Calculate the numerical value Using a calculator:

x ≈ 5.7735

**Answer:** The value of x is approximately 5.77 units.

4. Find Angle (Inverse Sine)

Given: A right triangle ABC (right angle at C) with opposite side AC = 5 units and hypotenuse AB = 13 units.
Calculate: The angle θB.

Step 1: Identify the trigonometric ratio We are given the opposite side and the hypotenuse. We use the inverse sine function (sin-1) to find the angle:

$sin(θ) = \cfrac{Opposite}{Hypotenuse}$

Step 2: Substitute values and apply inverse function

$sin(θ) = \cfrac{5}{13}$

$θ = sin^{-1}(\cfrac{5}{13})$

Step 3: Calculate the numerical value Using a calculator:

θ ≈ 22.62°

**Answer:** The angle θ is approximately 22.62°.

5. Find Angle (Inverse Cosine)

6. Find Angle (Inverse Tangent)

Given: A right triangle ABC (right angle at C) with opposite side AC = 4 units and adjacent side BC = 7 units.

7. Find Other Ratios (Given Sine)

Given: In a right triangle, $sin(θ) = \cfrac{3}{5}$.
Calculate: cos(θ) and tan(θ).

Step 1: Use the Pythagorean Identity We use the identity sin²(θ) + cos²(θ) = 1 to find cosine:

$(\frac{3}{5})² + cos²(θ) = 1$

$\frac{9}{25} + cos²(θ) = 1$

$cos²(θ) = 1 - \frac{9}{25} = \frac{16}{25}$

$cos(θ) = \frac{4}{5} $ (assuming a positive angle in a standard triangle)

Step 2: Calculate Tangent We use the identity $tan(θ) = \frac{sin(θ)}{cos(θ)}$:

$tan(θ) = \cfrac{\frac{3}{5}}{\frac{4}{5}} = \frac{3}{4}$

**Answer:** $cos(θ) = \frac{4}{5} \text{ , } tan(θ) = \frac{3}{4}$.

8. Find Other Ratios (Given Cosine)

Given: In a right triangle, $cos(θ) = \frac{5}{13}$.
Calculate: sin(θ) and tan(θ).

Step 1: Use the Pythagorean Identity We use the identity sin²(θ) + cos²(θ) = 1 to find sine:

$sin²(θ) + (\frac{5}{11})² = 1$

$sin²(θ) + \frac{25}{169} = 1$

$sin²(θ) = 1 - \frac{25}{169} = \frac{144}{169}$

$sin(θ) = \frac{12}{13}$

Step 2: Calculate Tangent We use the identity tan(θ) = sin(θ) / cos(θ):

$tan(θ) = \cfrac{\frac{12}{13}}{\frac{5}{13}} = \frac{12}{5}$

**Answer:** $sin(θ) = \frac{12}{13} \text{ , } tan(θ) = \frac{12}{5}$.

9. Find Other Ratios (Given Tangent)

Given: In a right triangle, $tan(θ) = \frac{8}{15}$.
Calculate: sin(θ) and cos(θ).

Step 1: Use the Pythagorean theorem to find the hypotenuse Let Opposite = 8 and Adjacent = 15.

Hypotenuse² = Opposite² + Adjacent²

Hypotenuse² = 8² + 15² = 64 + 225 = 289

Hypotenuse = $\sqrt{289} = 17$

Step 2: Define Sine and Cosine

$sin(θ) = \frac{Opposite}{Hypotenuse} = \frac{8}{17}$

$cos(θ) = \frac{Adjacent}{Hypotenuse} = \frac{15}{17}$

**Answer:** $sin(θ) = \frac{8}{17} \text{ , } cos(θ) = \frac{15}{17}$.

10. Find Height of a Building (Angle of Elevation)

Given: A right triangle scenario (building height as opposite side, distance as adjacent side) with an angle of elevation θ = 35° and adjacent distance = 100 ft.
Calculate: The height of the building (Opposite side x).

Step 1: Identify the trigonometric ratio We use the tangent function relating opposite (height) and adjacent (distance):

$tan(θ) = \frac{Opposite}{Adjacent}$

Step 2: Substitute values and solve for height x

$tan(35°) = \frac{x}{100}$

x = 100 × tan(35°)

Step 3: Calculate the numerical value Using a calculator:

x ≈ 70.02

**Answer:** The height of the building is approximately 70.02 ft.

3. Finding other trigonometric values

Solved Examples

Detailed solutions for 10 problems.

1. Given sin(x), find cos(x) and tan(x)

$ \text{Given: sin(x) = } \frac{1}{3}, \text{ x is acute.} $
$ \text{Calculate: cos(x) and tan(x).} $

Step 1: Use the Pythagorean Identity We use the identity sin²(x) + cos²(x) = 1:

$(\frac{1}{3})² + cos²(x) = 1$

$\frac{1}{9} + cos²(x) = 1$

$cos²(x) = 1 - \frac{1}{9} = \frac{8}{9}$

$cos(x) = \sqrt{\frac{8}{9}} = \frac{2\sqrt2}{3}$

(Since x is acute, cosine is positive.)

Step 2: Calculate Tangent We use the identity $tan(x) = \frac{sin(x)}{cos(x)}$:

$tan(x) = \cfrac{\frac{1}{3}}{\frac{2\sqrt2}{3}} = \frac{1}{2\sqrt2}$

Rationalize the denominator:

$tan(x) = \frac{\sqrt2}{4}$

**Answer:** $cos(x) = \frac{2\sqrt2}{3} \text{ , } tan(x) = \frac{\sqrt2}{4}$.

2. Given cos(x), find sin(x) and tan(x)

$ \text{Given: cos(x) = } \frac{3}{5}, \text{ x is acute.} $
$ \text{Calculate: sin(x) and tan(x).} $

Step 1: Use the Pythagorean Identity

$sin^2(x) + (\frac{3}{5})^2 = 1$

$sin^(x) + \frac{9}{25} = 1$

$sin^2(x) = 1 - \frac{9}{25} = \frac{16}{25}$

$sin(x) = \frac{4}{5}$

Step 2: Calculate Tangent

$tan(x) = \frac{sin(x)}{cos(x)} = \cfrac{\frac{4}{5}}{\frac{3}{5}} = \frac{4}{3}$

**Answer:** $sin(x) = \frac{4}{5} \text{ , } tan(x) = \frac{4}{3}$.

3. Given tan(x), find sin(x) and cos(x)

$ \text{Given: tan(x) = } \frac{5}{12}, \text{ x is acute.} $
$ \text{Calculate: sin(x) and cos(x).} $

Step 1:

$\frac{sin(x)}{cos(x)} = \frac{5}{12}$

$sin(x) = \frac{5}{12}cos(x)$

$ (\frac{5}{12}cos(x))^2 + cos^2(x) = 1$

$ \frac{25}{144}cos^2(x) + cos^2(x) = 1$

$ \frac{169}{144}cos^2(x) = 1$

$ cos^2(x) = \frac{144}{169}$

$ cos(x) = \sqrt{\frac{144}{169}} = \frac{12}{13} $

Step 2: Find Sine

$ \sin(x) = \tan(x) \times \cos(x) = \frac{5}{12} \times \frac{12}{13} = \frac{5}{13} $

**Answer:** sin(x) = $\frac{5}{13} \text{ , } cos(x) = \frac{12}{13}$.

4. Given sin(x), find others

$ \text{Given: sin(x) = } \frac{1}{2} \text{, x is acute.} $
$ \text{Calculate: cos(x), and tan(x).} $

Step 1: Find Cosine using Identity

$ \cos^2(x) = 1 - \sin^2(x) = 1 - (\frac{1}{2})^2 = 1 - \frac{1}{4} = \frac{3}{4} $

$ \cos(x) = \frac{\sqrt{3}}{2} $

Step 2: Find Tangent

$ \tan(x) = \frac{\sin(x)}{\cos(x)} = \cfrac{\frac{1}{2}}{\frac{\sqrt3}{{2}}} = \frac{1}{\sqrt{3}} $

$ \tan(x) = \frac{\sqrt{3}}{3} $

**Answer:** $ cos(x) = \frac{\sqrt{3}}{2} \text{ , } tan(x) = \frac{\sqrt{3}}{3}$.

5. Given cos(x), find others

$ \text{Given: cos(x) = } \frac{5}{13}, \text{ x is acute.} $
$ \text{Calculate: sin(x) and tan(x).} $

Step 1: Find Sine

$ \sin^2(x) = 1 - \cos^2(x) = 1 - (\frac{5}{13})^2 = 1 - \frac{25}{169} = \frac{144}{169} $

$ \sin(x) = \frac{12}{13} $

Step 2: Find tangent

$ \tan(x) = \frac{\sin(x)}{\cos(x)} = \cfrac{\frac{12}{13}}{\frac{5}{13}} = \frac{12}{5} $

**Answer:** $sin(x) = \frac{12}{13} \text{ , } tan(x) = \frac{12}{5}$.

6. Given tan(x), find others

$ \text{Given: tan(x) = } \sqrt{3}, \text{ x is acute.} $
$ \text{Calculate: sin(x) and cos(x).} $

Step 1:

$\frac{sin(x)}{cos(x)} = \sqrt{3}$

$sin(x) = \sqrt{3}cos(x)$

$ (\sqrt{3}cos(x))^2 + cos^2(x) = 1$

$ 3cos^2(x) + cos^2(x) = 1$

$ 4cos^2(x) = 1$

$ cos^2(x) = \frac{1}{4}$

$ cos(x) = \sqrt{\frac{1}{4}} = \frac{1}{2} $

Step 2: Find Sine

$ \sin(x) = \tan(x) \times \cos(x) = \frac{\sqrt{3}}{3} \times \frac{\sqrt{3}}{2} = \frac{3}{6} = \frac{1}{2} $

**Answer:** $sin(x) = \frac{1}{2} \text{ , } cos(x) = \frac{\sqrt{3}}{2}$.

7. Given sin(x), find cos(x) and tan(x)

$ \text{Given: sin(x) = } \frac{2}{5}, \text{ x is acute.} $
$ \text{Calculate: cos(x) and tan(x).} $

Step 1: Use the Pythagorean Identity

$ (\frac{2}{5})^2 + \cos^2(x) = 1 $

$ \frac{4}{25} + \cos^2(x) = 1 $

$ \cos^2(x) = 1 - \frac{4}{25} = \frac{21}{25} $

$ \cos(x) = \frac{\sqrt{21}}{5} $

Step 2: Calculate Tangent

$ \tan(x) = \frac{\sin(x)}{\cos(x)} = \cfrac{\frac{2}{5}}{\frac{\sqrt{21}}{5}} = \frac{2}{\sqrt{21}} $

$ \tan(x) = \frac{2\sqrt{21}}{21} $

**Answer:** $cos(x) = \frac{\sqrt{21}}{5} \text{ , } tan(x) = \frac{2\sqrt{21}}{21}$.

8. Given cos(x), find sin(x) and tan(x)

$ \text{Given: cos(x) = } \frac{1}{4}, \text{ x is acute.} $
$ \text{Calculate: sin(x) and tan(x).} $

Step 1: Use the Pythagorean Identity

$ \sin^2(x) + (\frac{1}{4})^2 = 1 $

$ \sin^2(x) + \frac{1}{16} = 1 $

$ \sin^2(x) = 1 - \frac{1}{16} = \frac{15}{16} $

$ \sin(x) = \frac{\sqrt{15}}{4} $

Step 2: Calculate Tangent

$ \tan(x) = \frac{\sin(x)}{\cos(x)} = \cfrac{\frac{\sqrt{15}}{4}}{\frac{1}{4}} = \sqrt{15} $

**Answer:** $sin(x) = \frac{\sqrt{15}}{4} \text{ , } tan(x) = \sqrt{15}$.

9. Given tan(x), find sin(x) and cos(x)

$ \text{Given: tan(x) = 2, x is acute.} $
$ \text{Calculate: sin(x) and cos(x).} $

Step 1: Use the definition of tangent in a right triangle $ \tan(x) = \frac{\text{opposite}}{\text{adjacent}} = \frac{2}{1} $. Let opposite = 2 and adjacent = 1.

$ \text{hypotenuse} = \sqrt{2^2 + 1^2} = \sqrt{5} $

Step 2: Find Cosine and Sine

$ \cos(x) = \frac{\text{adjacent}}{\text{hypotenuse}} = \frac{1}{\sqrt{5}} = \frac{\sqrt{5}}{5} $

$ \sin(x) = \frac{\text{opposite}}{\text{hypotenuse}} = \frac{2}{\sqrt{5}} = \frac{2\sqrt{5}}{5} $

**Answer:** $\sin(x) = \frac{2\sqrt{5}}{5} \text{ , } \cos(x) = \frac{\sqrt{5}}{5}$

10. Given sin(x) = 3cos(x), find tan(x), sin(x), and cos(x)

$ \text{Given: } \sin(x) = 3\cos(x), \text{ x is acute.} $
$ \text{Calculate: } \tan(x), \sin(x), \text{ and } \cos(x). $

Step 1: Find Tangent Divide both sides of the equation by $ \cos(x) $.

$ \frac{\sin(x)}{\cos(x)} = 3 $

Using the identity $ \tan(x) = \frac{\sin(x)}{\cos(x)} $, we get:

$ \tan(x) = 3 $

Step 2: Use the Right Triangle Method Since $ \tan(x) = \frac{3}{1} $, we can treat the opposite side as $ 3 $ and the adjacent side as $ 1 $.

$ \text{hypotenuse} = \sqrt{3^2 + 1^2} = \sqrt{10} $

Step 3: Calculate Sine and Cosine

$ \sin(x) = \frac{3}{\sqrt{10}} = \frac{3\sqrt{10}}{10} $

$ \cos(x) = \frac{1}{\sqrt{10}} = \frac{\sqrt{10}}{10} $

**Answer:** $ \tan(x) = 3, \sin(x) = \frac{3\sqrt{10}}{10}, \cos(x) = \frac{\sqrt{10}}{10} $

4. Trigonometric Identities

Proving Identities

Learn the strategies and practice proving trigonometric equations using only sin, cos, and tan.

Solved Examples (1-10)

1. Prove $(\sin x + \cos x)^2 + (\sin x - \cos x)^2 = 2$.

**Solution:** Expand binomials: $(\sin^2 x + 2\sin x\cos x + \cos^2 x) + (\sin^2 x - 2\sin x\cos x + \cos^2 x)$. Group terms: $(\sin^2 x + \cos^2 x) + (\sin^2 x + \cos^2 x)$. Apply $\sin^2 x + \cos^2 x = 1$: $1 + 1 = \mathbf{2}$.

2. Prove $\tan x \sin x + \cos x = \frac{1}{\cos(x)}$ (Simplified to tan/sin/cos only: $\frac{\sin^2 x}{\cos x} + \cos x = \frac{1}{\cos x}$).

**Solution:** Add fractions: $\frac{\sin^2 x}{\cos x} + \frac{\cos^2 x}{\cos x} = \frac{\sin^2 x + \cos^2 x}{\cos x}$. Numerator is 1: $\frac{1}{\cos x}$.

3. Prove $\frac{1 + \tan^2 \theta}{1} = \frac{1}{\cos^2 \theta}$ (using only tan/cos).

**Solution:** Use $1 + \tan^2 \theta = 1 + \frac{\sin^2 \theta}{\cos^2 \theta} = \frac{\cos^2 \theta + \sin^2 \theta}{\cos^2 \theta} = \frac{1}{\cos^2 \theta}$.

4. Prove $\sin^2 x = 1 - \cos^2 x$.

**Solution:** Start with $\sin^2 x + \cos^2 x = 1$. Subtract $\cos^2 x$ from both sides: $\sin^2 x = \mathbf{1 - \cos^2 x}$.

5. Prove $\cos \theta (\frac{1}{\cos \theta} - \cos \theta) = \sin^2 \theta$.

**Solution:** Distribute $\cos \theta$: $\frac{\cos \theta}{\cos \theta} - \cos^2 \theta = 1 - \cos^2 \theta$. Apply Pythagorean identity: $\mathbf{\sin^2 \theta}$.

6. Prove $\frac{\sin x}{1 + \cos x} + \frac{1 + \cos x}{\sin x} = \frac{2}{\sin x}$.

**Solution:** Add fractions: $\frac{\sin^2 x + (1 + \cos x)^2}{\sin x (1 + \cos x)} = \frac{2(1 + \cos x)}{\sin x (1 + \cos x)}$. Cancel $(1 + \cos x)$: $\frac{2}{\sin x}$.

7. Prove $(\frac{1}{\cos \theta} + \tan \theta)(\frac{1}{\cos \theta} - \tan \theta) = 1$.

**Solution:** Difference of squares: $\frac{1}{\cos^2 \theta} - \tan^2 \theta = \frac{1}{\cos^2 \theta} - \frac{\sin^2 \theta}{\cos^2 \theta} = \frac{1 - \sin^2 \theta}{\cos^2 \theta}$. Use $\cos^2 \theta = 1 - \sin^2 \theta$: $\frac{\cos^2 \theta}{\cos^2 \theta} = \mathbf{1}$.

8. Prove $\frac{\sin^2 x}{1 - \cos x} = 1 + \cos x$.

**Solution:** Start with numerator $\sin^2 x = 1 - \cos^2 x$. Factor numerator as $\frac{(1 - \cos x)(1 + \cos x)}{1 - \cos x}$. Cancel terms: $\mathbf{1 + \cos x}$.

9. Prove $\frac{\cos \theta}{\sin \theta} + \frac{\sin \theta}{\cos \theta} = \frac{1}{\sin \theta \cos \theta}$.

**Solution:** Find common denominator $\sin \theta \cos \theta$: $\frac{\cos^2 \theta + \sin^2 \theta}{\sin \theta \cos \theta}$. Numerator is 1: $\mathbf{\frac{1}{\sin \theta \cos \theta}}$.

10. Prove $(1 - \sin x)(1 + \sin x) = \cos^2 x$.

**Solution:** Multiply (difference of squares): $1 - \sin^2 x$. Use Pythagorean identity $1 - \sin^2 x = \mathbf{\cos^2 x}$.

**Solution:** Rewrite $\tan^2 \theta$: $\frac{\sin^2 \theta}{\cos^2 \theta} \cdot \cos^2 \theta + \sin^2 \theta = \sin^2 \theta + \sin^2 \theta = \mathbf{2\sin^2 \theta}$.
