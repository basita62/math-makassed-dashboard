Logarithmic and exponential functions ln(x) and exp(x)


  • $\frac{d}{dx}(\ln|x|) = \frac{1}{x}$ ; $\frac{d}{dx}\ln|u| = \frac{u'}{u}$
  • $\ln(a \times b) = \ln a + \ln b$, where $a > 0$ and $b > 0$
  • $\ln\left(\frac{a}{b}\right) = \ln a - \ln b$, where $a > 0$ and $b > 0$
  • $\ln\left(\frac{1}{a}\right) = - \ln a$, where $a > 0$
  • $\ln a^n = n \ln a$, where $a > 0$ and $n$ is real
  • $\ln 1 = 0$ ; $\ln e = 1$ ; $\ln e^2 = 2$
  • $\ln e^n = n$ ; $\ln \frac{1}{e} = -1$
  • $a = \ln e^a = e^{\ln a}$
  • $\ln x = a \iff x = e^a$
  • If $e^x = a$ then $x = \ln a$ $\thinspace$ where $a > 0$
  • $\ln a = \ln b \iff a = b$, where $a > 0$ and $b > 0$
  • $\ln a < \ln b \iff a < b$, where $a > 0$ and $b > 0$
  • $\ln a > \ln b \iff a > b$, where $a > 0$ and $b > 0$
  • $\lim\limits_{x\to 0^+} \ln x = -\infty$
  • $\lim\limits_{x\to +\infty} \ln x = +\infty$
  • $\lim\limits_{x\to 0^+} x \ln x = 0^-$
  • $\lim\limits_{x\to +\infty} \frac{\ln x}{x} = 0$
  • $\lim\limits_{x\to +\infty} \frac{x}{\ln x} = +\infty$
  • $\lim\limits_{x\to 0^+} x^n \ln x = 0^-$
  • $\lim\limits_{x\to +\infty} \frac{\ln x}{x^n} = 0^+$ where $n > 0$.
  • The function: $x \mapsto \ln x$ has domain = ]0; +$\infty$[ and range = ]$\infty$; +$\infty$[.
  • The function: $x \mapsto e^x$ has domain = ]$\infty$; +$\infty$[ and range = ]0; +$\infty$[.
  • These two functions are inverses of each other. Their graphs are symmetrical with respect to the first bisector ($y = x$).
  • $y = \ln x \iff x = e^y$
  • $e^a = e^b \iff a = b$
  • $e^a < e^b \iff a < b$
  • $e^a > e^b \iff a > b$
  • $e^a \cdot e^b = e^{a+b}$
  • $\frac{e^a}{e^b} = e^{a-b}$
  • $(e^a)^b = e^{ab}$
  • $e^{-a} = \frac{1}{e^a}$
  • $\frac{d}{dx}(e^x) = e^x \text{ and } \frac{d}{dx}(e^u) = e^u \cdot u'$
  • $\lim\limits_{x\to -\infty} e^x = 0^+$
  • $\lim\limits_{x\to +\infty} e^x = +\infty$
  • $\lim\limits_{x\to +\infty} \frac{e^x}{x} = +\infty$
  • $\lim\limits_{x\to +\infty} \frac{x}{e^x} = 0$
  • $\lim\limits_{x\to 0} \frac{e^x-1}{x} = 1$
  • $\lim\limits_{x\to +\infty} e^{-x} = 0$
  • $\lim\limits_{x\to +\infty} \frac{e^x}{x^n} = +\infty$
  • $\lim\limits_{x\to -\infty} x^n e^x = 0$ , ($n \in \mathbb{Z}^+$)

Derivatives and Antiderivatives rules

  • L'Hopital's Rule for limits of indeterminate forms $\frac{0}{0}$ or $\frac{\infty}{\infty}:$
  • If $\lim\limits_{x\to a} \frac{f(x)}{g(x)}$ exists, then $\lim\limits_{x\to a} \frac{f'(x)}{g'(x)}$ exists, then $\lim\limits_{x\to a} \frac{f(x)}{g(g)} = \lim\limits_{x\to a} \frac{f'(x)}{g'(x)}$
  • $F(x)$ is an anti-derivative $G(x) \iff F'(x) = G(x)$
  • $F(x) = \displaystyle\int G(x)dx \iff F'(x) = G(x)$.
  • $\frac{d}{dx} (\displaystyle\int f(x)dx) = f(x)$ and $\displaystyle\int f'(x)dx = f(x) + C$
  • $\displaystyle\int x^n dx = \frac{x^{n+1}}{n + 1} + C$ ($n \neq -1$)
  • $\displaystyle\int(u + v)dx = \displaystyle\int udx + \displaystyle\int vdx$ ; $\displaystyle\int kudx = k \displaystyle\int udx$ ; $k \in \mathbb{R}$
  • $\displaystyle\int u'v = uv - \displaystyle\int v'u$ (integration by parts)
  • $\displaystyle\int u'f(u)dx = \displaystyle\int f(u)du$
  • $f_x' = f_u' \times u_x'$ (chain rule: $\frac{df}{dx} = \frac{df}{du} \times \frac{du}{dx}$)
  • $(u \pm v)' = u' \pm v'$
  • $(uv)' = u'v + v'u$
  • $\left(\frac{u}{v}\right)' = \frac{u'v - v'u}{v^2}$
  • $\left(\frac{1}{u}\right)' = - \frac{u'}{u}$

Definite Integrals and Applications

  • If $F$ is an anti-derivative of $f$ over $I$, then $\displaystyle\int_a^b f(x)dx = F(b) - F(a)$.
  • $\displaystyle\int_a^b f(x)dx$ = algebraic area of the domain limited by the representative curve of $f$, the x-axis and the two straight lines of equations $x = a$ and $x = b$.
  • Mean value of a function (average value) over an interval $[a; b]$ is $\bar{f} = \frac{1}{b-a}\displaystyle\int_a^b f(x)dx$.
  • $\displaystyle\int_a^b (f(x) - g(x))dx$ = area of the domain limited by the representative curves of $f$ and $g$ where $f(x) > g(x)$ over $[a, b]$ and the two straight lines of equations $x = a$ and $x = b$.
  • $\displaystyle\int_b^a f(x)dx = - \displaystyle\int_a^b f(x)dx$
  • $\displaystyle\int_a^b f(x)dx = \displaystyle\int_a^c f(x)dx + \displaystyle\int_c^b f(x)dx$
  • $\displaystyle\int_a^b k dx = k(b - a)$ and $\displaystyle\int_a^b dx = b - a$ and $\displaystyle\int_a^a dx = 0$.
  • If $f(x)$ is an odd function, then $\displaystyle\int_{-a}^a f(x)dx = 0$, $a \in \mathbb{R}$.
  • If $f(x)$ is an even function, then $\displaystyle\int_{-a}^a f(x)dx = 2 \times \displaystyle\int_0^a f(x)dx$, $a \in \mathbb{R}$.
  • If $f(x) \leq g(x)$, then $\displaystyle\int f(x)dx \leq \displaystyle\int g(x)dx$
  • If $H(x) = \displaystyle\int_a^{u(x)} f(t)dt$, then $H'(x) = f(u(x)) \times u'(x)$, $a \in \mathbb{R}$

Special economics

  • Given the cost function : $C_T(x)$, then marginal cost function : $C_m(x) = C_T'(x)$
  • Given marginal cost function : $C_m(x)$, then total cost function : $C_T(x) = \displaystyle\int C_m(x)dx + K$.
